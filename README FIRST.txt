WHAT'S THIS APPLICATION ABOUT?

Description:

This is an Angular 17 UI TEST made with Karma and Jasmine.


--------------------------------------------------

HOW TO RUN IT LOCALLY?

--- RUN ANGULAR 17 APP SERVER:

1- Download the GITHub repository "UI KARMA_JASMINE".

2- Create the folder C:\UI KARMA_JASMINE and install the repository inside this folder.

3- Go to this folder with the Windows Explorer and type CMD in the address bar overwritting
    the path. Hit enter.

4- In the command prompt console type in "npm install"

5- After the installation of packages is finished, type in "ng test".
    If succeeded you should see a new google Chrome browser window, showing you
    a screen with a "Karma" screen with information about UI Unit Tests status.
